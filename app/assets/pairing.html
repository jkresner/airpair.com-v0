At airpair, our primary focus is promoting collaboration & knowledge share amongst developers. We believe pair programming is a powerful & critical tool to foster greater learning, faster development & better products. Thus we run a free virtual meetup for developers interested in pairing with & learning from each other.

{ join meetup group }

How our virtual meetups work?

24 hours before each meetup event, we send instructions in a meetup message with variations of each step below.

Our meetup is about experimenting on the best ways to enable developers to find pair programming help in real-time. The process might change from time to time. We'd love your input, so email us feedback & ideas.

{ 1. Pre-event post }

On the day, fill out a basic google form with the tech and challenge(s) you'd like to pair on.

{ 2. Join the event chatroom a.k.a IRC channel }

Connect to our irc channel

{ 3. Share a link to join chatroom }

"I just joined the remote pair programmers chat-room for their two hour weekly week up to find baller coders to pair with."

{ 4. Post your hello in the irc channel }

Hello!!! {option funny callout message}. I'm {full name}. I'm a {beginner|intermediate|advanced} with {technology(s)}. I've been developing for {n} years (or months). I would like to pair up on {insert what you're up for}.

{ 5. Create a google hangout and share the link }

* If joining someone else's hangout you can ignore these steps.

Open google+ and click "start a hangout"

{ 6. Give feedback }

We will send you a questionnaire asking for feedback on the success of your pairing session as well as feedback on your pairing partner(s).
