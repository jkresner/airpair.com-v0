{ Remote pair programmers meetup }

At airpair, our primary focus is promoting collaboration & knowledge share amongst developers. We believe pair programming is a powerful & critical tool to foster greater learning, faster development & better products. We run a free virtual meetup for developers interested in pairing with & learning from each other.

{ join meetup group }

How our virtual meetups work?

Our meetup is an experiment on uncovering the best ways to enable developers to find pair programming help. The process will change as we go. We'd love your input, please email us your feedback & ideas.

24 hours before each meetup, we send instructions in a meetup message with variations of the following steps.

{ 1. Pre-event post }

Before the meetup, post the technologies & things you'd like to pair on..

{ 2. Join the event chatroom a.k.a IRC channel }
