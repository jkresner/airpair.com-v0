How our virtual meetup works

Our meetup is an experiment on uncovering the best ways to enable developers to find pair programming help. We'd love your input, please email us your feedback & ideas.

24 hours before each meetup, we'll send instructions in a meetup message with variations of the following steps.

{ 1. Pre-event post }

Before the meetup, post the technologies & things you'd like to pair on..

{ 2. Join the event chatroom a.k.a IRC channel }

At the scheduled time, connect to our #pariprogrammers irc channel

{ 3. Help increase attendance. }

As you join our chartroom share that you're virtually pair programing by clicking the share icons here:


{ 4. Find & message those you'd like to pair with}

Use the google spreadsheet to see who uses the same technologies & what everyone is interested in working on. Then start a conversation to see if it's a good fit.

{ 5. Create & invite pairs to a google hangout }

Once you've found one or more pairs, ONE of you must click the buttons below to create a hangout.

You have the option to create a public broadcasted hangout, or a private hangout.

Create recorded hangout Create private hangout

Once in the hangout, use get your pairs google accounts email address and invite them to the hangout. From there you can try with pure screensharing, or optionally agree on collaborative text editing tools.

{ 6. Give feedback }

The day after the meetup, we'll send a questionnaire asking for feedback of your experience as well as feedback for your pairing partner(s).
